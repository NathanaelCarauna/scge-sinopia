## Métodos Alternativos para Gerar Imagens

### 1. 🌐 **Online - Mermaid Live Editor**
1. Acesse: https://mermaid.live
2. Cole o conteúdo do arquivo `diagrama_mermaid.mmd`
3. Clique em "Download PNG" ou "Download SVG"

### 2. 🌐 **Online - PlantUML Server**
1. Acesse: http://www.plantuml.com/plantuml/uml
2. Cole o conteúdo do arquivo `diagrama_casos_uso_oe2.puml`
3. Clique em "Submit" e depois "PNG" ou "SVG"

### 3. 💻 **Local - Mermaid CLI (já instalado)**
```bash
# Gerar PNG
mmdc -i diagrama_mermaid.mmd -o output.png -w 1920 -H 1080 --backgroundColor white

# Gerar SVG
mmdc -i diagrama_mermaid.mmd -o output.svg --backgroundColor white

# Gerar PDF
mmdc -i diagrama_mermaid.mmd -o output.pdf --backgroundColor white
```

### 4. 💻 **Local - PlantUML (necessita instalação)**
```bash
# Instalar PlantUML
sudo apt install plantuml  # Ubuntu/Debian
# ou
brew install plantuml      # macOS

# Gerar imagem
plantuml diagrama_casos_uso_oe2.puml
```

### 5. 🔧 **VSCode Extensions**
- **Mermaid Preview**: Preview e export direto no VSCode
- **PlantUML**: Preview e export de diagramas PlantUML
- **Draw.io Integration**: Para edição visual

### 6. 🖼️ **Conversão para outros formatos**
```bash
# Converter PNG para JPG
convert diagrama_casos_uso_oe2.png diagrama_casos_uso_oe2.jpg

# Converter PNG para PDF
convert diagrama_casos_uso_oe2.png diagrama_casos_uso_oe2.pdf

# Redimensionar
convert diagrama_casos_uso_oe2.png -resize 50% diagrama_casos_uso_oe2_small.png
```
